# README

## Как запустить / открыть верстку

    - Откройте файл `index.html` в браузере.
    - В проводнике дважды кликните `index.html` или используйте `Открыть файл` в браузере.


## Страницы / маршруты

- `index` — главная страница
- `indexTaged`— главная страница, с применённым тегом
- `question` — страница вопроса
- `ask` — страница создания вопроса
- `login` — страница входа
- `signup` — страница регистрации   
- `profile` — страница настроек профиля
- `base` — шаблон страницы (для неавторизованного пользователя)
- `baseAuthed` — шаблон страницы (для авторизованного пользователя)
